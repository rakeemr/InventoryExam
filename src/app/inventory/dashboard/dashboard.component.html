Clientes Este modulo posee todo lo que conlleva el CRUD de la parte de clientes de nuestro sistema. Apenas se entre se encontrará con una lista que contiene a todos los clientes agregados hasta el momento cargados desde la base de datos por medio del Backend, sino existen clientes la lista se encontrará vacía. También contiene un botón de agregar y otros para editar y/o eliminar a alguno de los clientes.

Apenas se entre se encontrará con una lista que contiene a todos los productos agregados hasta el momento cargados desde la base de datos por medio del Backend, sino existen productos la lista se encontrará vacía. También contiene un botón de agregar y otros para editar y/o eliminar a alguno de los productos.
Inventario Este modulo posee todo lo que conlleva el CRUD de la parte de inventario de nuestro sistema. Apenas se entre se encontrará con una lista que contiene todo el inventario agregado hasta el momento cargado desde la base de datos por medio del Backend, sino existe nada en inventario la lista se encontrará vacía. También contiene un botón de agregar y otros para editar y/o eliminar alguno de los elementos del inventario.
Facturación En este modulo se encontraran las facturas asociadas a un cliente con sus respectivos productos. En caso de que algún cliente desista de su compra, la aplicación en este apartado logrará desabilitar la factura creada.
